# GPA/CGPA Calculator

A responsive GPA/CGPA Calculator built with HTML, CSS, and JavaScript.

## Features
- Calculate Semester GPA
- Track Running CGPA
- Add subjects with credits and grades
- What-If Mode for future semester simulation
- Responsive UI
- Easy to use

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Use
1. Add subjects with credits and grades.
2. Click Calculate GPA.
3. Enter previous CGPA details to calculate running CGPA.
4. Use What-If Mode to estimate your graduation CGPA.
